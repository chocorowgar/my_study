git

Is a Distributed Version Control System(VCS)

Repository(repo) (儲存庫)A storage space that contains all the files in your project and a history of changes
Clone Copies a remote repository to your local machine
Pull Fetches changes from a remote repository and merges them into your local branch
Push Sends your local commits to a remote repository
Commit Records changes to the repository. Just like a snapshot
Branch (分支)A branch is an independent line of development in a repository
Merge Combine 2 different branches
Staging An intermediate step between editing files and committing changes

Git Status

Stages of a File in Git

Untracked Files that are in your working directory, but not added to the repository
Tracked Files that Git knows about and are added to the repository
Modified A file that has been edited but not staged
Staged A file that has been added to the staging area and is ready to be committed
Committed(i.e. Unmodified yet) Changes that are saved in the repository
